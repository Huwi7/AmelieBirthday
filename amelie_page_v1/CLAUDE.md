## Project Overview

A minimal static HTML project. The entire project is a single file: `index.html`, which renders a centered "Hello, World!" page with basic CSS.

## Development

No build step, package manager, or framework is involved. Open `index.html` directly in a browser to preview, or serve it with any static file server:

```sh
python3 -m http.server
# or
npx serve .
```
